2024年

9月10日

iOS 版 SDK を更新し、macOS Sonoma 14.6.1 の Xcode 15.4 でビルドして実行できるようにしました。Xcode 15.3 以降では incompatible function pointer types に関するエラーが発生するようになったため、該当箇所を以下のように修正してエラーを抑制しました。

#pragma clang diagnostic push
#pragma clang diagnostic ignored "-Wincompatible-pointer-types"
// error code
#pragma clang diagnostic pop

9月6日

Android 版 SDK を更新し、compileSdk と targetSdk を 35 にしました。
